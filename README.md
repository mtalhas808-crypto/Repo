# Hasan Irfan - Premium Perfumes Website

A modern, responsive website for Hasan Irfan's premium perfume business.

## Features

- **Beautiful Hero Section** - Eye-catching landing page with smooth animations
- **Product Catalog** - Showcase of premium perfumes with filtering by category (Men, Women, Unisex)
- **Shopping Cart** - Full cart functionality with persistent storage
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Contact Form** - Easy way for customers to get in touch
- **Modern UI/UX** - Clean, elegant design with smooth transitions

## Getting Started

Simply open `index.html` in your web browser. No build process or dependencies required!

## File Structure

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - JavaScript functionality for cart, filtering, and interactions
- `README.md` - This file

## Customization

### Products
Edit the `products` array in `script.js` to add, remove, or modify products.

### Colors
Modify the CSS variables in `styles.css` under `:root` to change the color scheme:
- `--primary-color` - Main dark color
- `--secondary-color` - Accent/gold color
- `--accent-color` - Light background color

### Contact Information
Update the contact details in the `#contact` section of `index.html`.

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge).
